# LinkedIn Like Project version 4

This is a personal pet project to practice with HTML/CSS (TailwindCSS) and Javascript/Typescript to simulate the LinkedIn-like mobile web UI. Currently, the app is implemented only for the LinkedIn Profile page. The app has been developed from version [1](https://github.com/bi151103/linkedin-like-project), [2](https://github.com/bi151103/linkedin-like-project-v2), [3](https://github.com/bi151103/linkedin-like-project-v3) and now is at version [4](https://github.com/bi151103/linkedin-like-project-v4).

## Link of the deployed app

Deployed application:

## Tech stack

- HTML
- Tailwind CSS
- Javascript/TypeScript

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Run Tailwind in watch mode:

```bash
npm run dev
```

3. Run TypeScript in watch mode (in another terminal):

```bash
npm run dev:compile-ts
```

4. Open `index.html` using Live Server or your browser. Refer to [How do you set up a local testing server?](https://developer.mozilla.org/en-US/docs/Learn_web_development/Howto/Tools_and_setup/set_up_a_local_testing_server) to know how to set up a server in the local machine.
